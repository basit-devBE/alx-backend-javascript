## Introduction to Classes

JavaScript classes are a syntactical sugar over the existing prototype-based inheritance. They provide a more familiar and intuitive syntax for creating objects and defining their behavior.

## Creating a Class

To create a class in JavaScript, you can use the `class` keyword followed by the class name. Inside the class, you can define properties and methods.
